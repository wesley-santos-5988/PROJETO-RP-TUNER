# RP-Tuner

Projeto universitário simples para controle e afinação de parâmetros em dispositivos Raspberry Pico.

## Descrição

Este repositório contém código e documentação do projeto RP-Tuner, desenvolvido como trabalho acadêmico. O objetivo é criar uma interface mínima para ajustar parâmetros de tempo e desempenho em aplicações embarcadas com Raspberry Pi Pico.

## Estrutura do projeto

- src/: código-fonte do projeto
- docs/: documentação adicional
- examples/: exemplos de uso

## Requisitos

- Raspberry Pi Pico (ou Pico W)
- MicroPython ou C/C++ SDK instalado
- Ferramentas de desenvolvimento (VS Code, build tools)

## Instalação

1. Clone o repositório:

	git clone <URL-do-repositório>

2. Siga as instruções na pasta `docs/` para configurar o ambiente e gravar o firmware no Pico.

## Uso

1. Conecte o Raspberry Pi Pico ao computador.
2. Carregue o firmware ou os scripts em `src/` para o dispositivo.
3. Use os exemplos em `examples/` para testar ajustes de parâmetros.
